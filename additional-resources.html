Additional Resources¶

This page provides some additional resources for further reading after the Workshop which may be of interest.

MIMIC¶

Further information on the MIMIC Database is available at:

  • The MIMIC Database homepage

  • A brief introduction to MIMIC on the Laboratory for Computational Physiology’s website

  • A textbook on MIMIC

  • The article describing MIMIC-III

The WFDB Specifications and Toolbox¶

  • WFDB Specifications: Documentation of the specifications for Waveform Database (WFDB) files and concepts.

  • WFDB Software Package: The original software package written in C.

  • WFDB Python Package

  • WFDB Toolbox for Matlab

Open Research¶

  • The Turing Way handbook to reproducible, ethical and collaborative data science, which includes guidelines on Open Research.

  • The Software Sustainability Institute, including Guides on ensuring software sustainability.

Software development and management¶

  • An introduction to Python is available here.

  • Tutorials on Git and GitHub, which are used for version control and collaborative code development, are available here and here.

Cuffless Blood Pressure estimation¶

  • Review of Cuffless Blood Pressure Measurement: a review of the topic by leaders in the field.

  • Evaluating Cuffless Blood Pressure Devices: a review on “the capabilities and limitations of emerging cuffless BP measurement devices”, with proposals of how to evaluate such devices.

Photoplethysmography¶

  • Photoplethysmography Signal Processing and Synthesis: a textbook chapter providing a comprehensive overview of PPG signal processing.

  • Wearable photoplethysmography for cardiovascular monitoring: a review paper detailing different aspects of wearable photoplethysmography, including signal processing and clinical applications.

  • Establishing best practices in photoplethysmography signal acquisition and processing: an article discussing whether it would be possible and beneficial to establish best practices for photoplethysmography signal acquisition and processing.

Datasets containing physiological signals¶

  • Physionet: PhysioNet hosts many datasets containing physiological signals, listed here.

  • Photoplethysmography datasets: There are several publicly available datasets containing photoplethysmogram signals, many of which are listed here.
